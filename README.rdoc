SAutoC - Multi-Model Autocomplete Search Box (using Redis and Soulmate)

Requirements:

Rails 3.2.16

Ruby 2.0.0

Redis

Postgres

Setup:

Clone this app to your desired location

Bundle Install

Configure database.yml

rake db:create:all

rake db:migrate

Start Redis Server using “redis-server” command

rake db:seeds

Start Rails Server using “rails server” command
