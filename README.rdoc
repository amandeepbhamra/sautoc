SAutoC - Multi-Model Autocomplete Search Box (using Redis and Soulmate)
=============================================================================

Requirements: 

1. Rails 3.2.16
2. Ruby 2.0.0
3. Redis
4. Postgres 

Setup:

1. Clone this app to your desired location
2. Bundle Install
3. Configure database.yml
4. rake db:create:all
5. rake db:migrate
6. Start Redis Server using "redis-server" command 
7. rake db:seeds
8. Start Rails Server using "rails server" command
